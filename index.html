Deploy the game folder, not this file

Netlify needs index.html at the root of your upload. This repo keeps the game in public/.

  1. Open a terminal in this project
  2. Run npm run prepare:deploy
  3. Open the deploy folder
  4. Zip everything inside deploy (not the parent project folder)
  5. Upload that zip to Netlify

Or connect this repo to Netlify with Git — netlify.toml runs the build for you.
